お仕事内容

エルカミーは、クライアントが抱える膨大なデータを活用し、様々な課題解決と意思決定支援を提供する会社です。クライアントのDX化促進のため、Google Cloud™ で最適なデータ活用・MLOpsの実現を支援しています。
コンサルティングを通してクライアントの本当の課題を導き出し、解決まで伴走することが私たちが提供しているサービスです。

エルカミーでは、ただいまデータサイエンティストを目指したい方を募集しています。クラウドを使った顧客のDX化促進や、内製化支援に興味がある方お待ちしています。

業界未経験の方も、データサイエンティストの補佐として分析やリサーチを中心に経験を積み、キャリアアップを目指せるポジションです。

責任ある行動が重んじられる環境ですが、自由な発想と主体性を活かせるフラットな組織文化です！

◾️フルリモートです
Notion, Slack, Gather等を使用して常にコミュニケーションをとっています。リモート会議が多いのでオンラインでも活発にコミュニケーションをにとれる方が活躍できる職場です。

◾️お仕事について
主にリサーチと分析を中心とした業務です。データサイエンティストの補佐業務をはじめ社内のR&D活動、その他バックオフィス関連業務を中心にトレーニングを積み、将来的にプロジェクトの主力メンバーとして活躍できるレベルを目指していただきます。

◾️案件例
メーカーの売り上げ・販売予測や、ECサイトの業務内容改善案件、金融機関におけるシステム開発など、プロジェクトの業界は多岐に渡ります。

◾️Notion使っています
Elcamyでは、作業のアウトプットやノウハウ共有等、Notionをベースに作業を行うことが多いです。Notionに使い慣れている方大歓迎です。
