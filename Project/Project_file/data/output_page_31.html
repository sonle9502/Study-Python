■業務内容の詳細

  • 防災・インフラメンテナンス分野などの官公庁事業を対象としたデジタルソリューションの企画立案・開発

    ＜例＞3次元空間情報・画像解析・センシング・データ分析〜活用などに係る開発、業務利用の推進、など
