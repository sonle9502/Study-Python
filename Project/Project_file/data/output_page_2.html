仕事内容

弊社では、Power BIというビジネスインテリジェンスツールを使って企業のデータ分析、可視化、レポート作成を支援するデータサイエンティストを募集しております。
これは、データドリブン経営により競争力を高めるために欠かせない役割であり、DX化に取り組む企業には必須のスキルで注目されています。Power BIを使ってビジネス上の課題を解決し、企業の成長に貢献することが期待されます。

弊社では、若年層にも公平にチャンスを与えるため、アルバイトからスタートしていただき、成果を上げていただければ社員に昇格していただけます。もちろん、学歴や経験にこだわることはありません。あなたの熱意やポテンシャルを重視し、最適な人材を選考します。

これからの日本を考え、若い方が活躍する機会を与えたいとおもっております。就業機会に恵まれない若年層の方々が、手に職を付けて弊社で活躍していただくため。その第一歩としての位置づけであります。そのため、 学生在籍期間での学生アルバイトとしての採用はしておりませんので、予め、ご理解ください。

選考は厳正かつ公正に行います。書類選考の後、面接、適性検査を行い、最終的にはあなたの人柄やスキル、やる気を総合的に評価し、採用を決定します。

データ分析は、専門的な知識が必要とされる分野です。しかし、未経験の方でも、当社で用意するトレーニングやPower BIトップセミナー講師のサポートにより、スキルアップを図ることができます。もちろん、あなた自身が学び、成長する姿勢が大切です。

１日３時間以上、週３～４回以上の勤務をお願いしています。弊社のビジョンに共感し、共に成長していける方のご応募をお待ちしています。

当社は中小企業に対して、データ分析力を提供することで、働く人々のエンパワーメントを促し、組織の強みを最大限に引き出していくことを目指しています。ビッグデータ分析、企業データ分析、Business IntelligenceツールやRPAツールなどを用いて、企業のDX（デジタルトランスフォーメーション）やデータドリブン経営などをサポートし、業務改善の活用コンサルティングを提供しています。

当社では、Power BIという分析ツールを使って、初心者から研修し、きちんとデータ分析が業務や経営の中で活用できるように設計し、コンサルティングしたり、企業の分析レポートを作成したりする仕事をしています。Power BIを使って、企業を元気にし、そこで働く人々にデータ分析という新しい力を与えることが私たちの目標です。

私たちは、少人数でコツコツと仕事をしてきましたが、ご依頼いただくことが多くなってきたため、一緒に日本を元気にしていく仲間を募集することにしました。ITエンジニアの経験は必要ありません。Excelで集計や分析を行ったことがあればOKです。そして、大切なのは、データ分析を使って、仕事や業務や経営を変えていきたいというマインドがある方です。当社は常にチャレンジ精神を持ち、失敗を恐れずに新しいことにチャレンジしています。指示待ちの人や受け身の人は求めていません。一緒にチャレンジし、経験を共有して成長していける方と一緒に働きたいと思っています。

私たちは、ベンチャー企業であり、常に変化を追求しています。どんどん変化し、いろんなことにチャレンジし、成長していきます。私たちと一緒に働いて、日本のビジネス界を変えていきましょう！
