【Visaプリペイド『バンドルカード』のアプリとカードを運営する、注目のBtoCフィンテックベンチャー企業です】
クレジットカードと同じようにVisaを使える店ならばどこでも支払いができるチャージ式のカードス『バンドルカード』を提供をしております。カードに並び、アプリでも提供をしております。クレジットカードを持てない学生や主婦層、クレジットカ―ドでの支払いに抵抗があるユーザにアプローチする。規制の多い金融業界の常識を打ち破り、「世の中のお金の流れを円滑にする」を企業理念に運営に取り組んでいる。

【会社の特徴】:
【アプリとリアルカード発行という二段構えを取っております】

【プリペイドカードで事業を行う意味】
与信が必要なクレジットカードは、高校生までは持てません。クレジットカードを持てない人や持ちたくない人等、世代や国籍を超えたあらゆる人々に、カード決済を体験してもらうことを目的としています。これからの日本ではキャッシュレス社会を目指しておりますが、クレジットカードのみの世界では、対応ができなくなる人に物理的に制限があります。同社はそんな不均衡な世界の解消を目指しております。事また、前にチャージした金額以上は使えないという点で、現金と似た感覚で買い物ができます。さらに購入履歴をすぐに確認できるので、お金の流れが見えやすく、使い過ぎを防げるメリットもあり、与信が必要ないので即時発行が可能です。
