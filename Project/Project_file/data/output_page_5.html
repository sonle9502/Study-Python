私たちについて

当社では医療機関経営の最重要課題における集患・人材などをはじめ、
医療機関がアナログのまま進んでしまっている部分のDX化を促進することで医療機関経営を効率化していきます。

DX化が遅れている医療業界に対して、弊社では経営的な視点をもって医療機関のDX化を支援し、
医療業界全体に革新を起こすような企業を目指しています。

【事業内容】
・医療機関専門WEB制作
・医療DX・WEBコンサルティング
・医療WEB広告運用
・医療SEO対策・サイト収益化
・医療機関専門動画制作
・代理店事業
