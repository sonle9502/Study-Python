職務内容詳細

採用 情報

経済学を応用したコンサルティング

NERA東京事務所では、独 禁 法と競争政策、知的財産、証券・金融、国際移転価格、国際仲裁、複雑な商事紛争に関わる規制や訴訟、エネルギー・ヘルスケア・テレコム等の規制産業における企業へのアドバイスやコンサルテーションの分野において、経済学関連の知識（ミクロ経済学、計量経済学、統計学、ファイナンス、会計、サーベイリサーチ等）を活用して実践的な解決策を提案できる 人材 を 募集 しています。

仕事内容

上記のプラクティス領域に関わる、様々な調査、計量経済学・統計学分析、価値評価（損害額を含む）、報告書・意見書作成、専門家証言等が仕事の内容になります。

調査や経済分析として行う分析内容は極めて広範にわたります。計量経済学や統計学を用いたデータ分析、データ分析に基づく損害額や価値評価、DCF等の手法による価値評価や損害賠償額の算定、ミクロ経済学やゲーム理論等に基づく理論分析、様々なデータソースからの検索・情報入手、アンケート調査の実施など、必要に応じて、様々な調査・分析作業に携わることになります。

海外の訴訟や規制にかかわる仕事に携わることも多く、クライアントが海外の企業であることも珍しくありません。日本語だけでなく英語でのコミュニケーション能力（読むだけでなく、執筆能力）も一定程度必要です。

募集 職種

マネジングディレクター／ディレクター／アソシエイトディレクター

アカデミックアフィリエイト

マネジングディレクター／ディレクター／アソシエイトディレクターは、チームの管理者としてのプロジェクトをリード、統括し、クライアントに直接対応していく役割を果たします。シニアコンサルタント／コンサルタントは、チームの一員として、一定程度自立的に判断してプロジェクトの中心的な業務をこなし、クライアントに提出する成果物を構成することが望まれます。将来的にはプロジェクトを管理し、プロジェクトを獲得する役割を担うことも期待されています。リサーチャー／アソシエイトアナリスト／アナリスト／シニアアナリストは、コンサルタントより上のランクのエコノミストの指示に従いながら様々な調査・分析に関与していくことになります。

アカデミックアフィリエイトは、大学等の研究機関における籍はそのままで、基本的にプロジェクトベースでNERAのメンバーと一緒に企業向けにコンサルティングを行うことを目的とするポジションです。

  • ミクロ経済学（ゲーム理論・産業組織論）、計量経済学・統計学、金融経済学、企業財務、会計、社会調査等の広い意味での経済学関連分野の専門知識を有する方 （特に国内外での関連分野の博士・修士号取得者歓迎）
  • 会計、ファイナンス関連の資格保有者歓迎（例えば、CPA）
  • コンサルティングファーム、シンクタンク、大学、研究機関、規制当局、金融機関、民間企業の企画・調査・経理・財務等で、NERAの業務に関連する実務経験を有する方を歓迎
  • 日本語／英語能力 （両言語でのレポート執筆、海外顧客とのコミュニケーションに対応できる英語力）
  • PCスキル（エクセル、ワード、パワーポイント） ※SAS/STATA/R等の統計・数量解析ソフトの使用経験尚可
