ＹＫＫ ＡＰは私たちがつくるArchitectural Products（建築用工業製品）を通して、社会を幸せにする会社でありたいと考えています。

そしてアルミの知識や技術を用いて高品質なアルミ形材を提供する産業製品。私たちの商品は、世界各地の気候・風土・文化にあわせ、そこで暮らす人々に健康で快適、安全で安心な生活を提供するとともにこれからの時代にふさわしい事業価値を創造します。
