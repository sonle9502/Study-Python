https://www.comico.jp/

定時内にて対応できる業務量を調整している為、残業がほとんどない方もいらっしゃいます。ワークライフバランスを整えたい方にもマッチするポジションで、お子さんがいらっしゃる子育て世代の方が多いのも特徴です。

NHNグループは日本、韓国、中国、タイ、ベトナム、シンガポール、アメリカにグループ会社をもつグローバルカンパニーとしてインターネットを通じた幅広い事業展開を行っております。社名のNHNとは「Next Human Network=インターネットに広がる無限の可能性を活かし人と人とのつながりと豊かな社会の実現」という想いが込められております。今後も世界市場で競い合うことのできる品質とテクノロジーの向上を図り、積極的に人材の採用を行っていく予定です。是非私たちと一緒に働きましょう。
