仕事内容

データアナリストとしてのスキルを磨けます。
位置情報を用いた最新のデジタルマーケティング、デジタル広告の効果最大化に向けたコンサルティングや分析、プロモーションの
改善提案など幅広く対応いただきます。
案件の進行管理、広告運用改善のディレクション、分析並びにレポーティング業務を主に担当していただきます。
※シニアレベルで即戦力として稼働できる方

詳細

◆案件の進行管理：クライアント様や広告代理店とのコミュニケーション、効果改善に向けた運用調整や分析提案のディレクション
◆コンサルティング：位置情報を活用したプロモーション発展の為の深い分析(デジタル広告運用経験やデジタルマーケティング経
験を活かした分析など)
◆SBグループのZ holdings(Yahoo! JAPAN,LINE)のビックデータを活用した新たなマーケティング施策の提案

お仕事の魅力

デジタル人材として最先端な領域でデータやプロモーションに触れることができ、総合力が試される仕事となるため、業務を通して
体得できるスキルがとても濃厚です。
