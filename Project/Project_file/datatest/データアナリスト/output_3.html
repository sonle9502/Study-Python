アピールポイント

＜募集背景＞

組織強化のための増員となります。

風音屋はIT系コンサルティング会社です。大手上場企業からスタートアップまで、幅広いクライアントのデータ分析やシステム構築を支援しています。

国立大学の研究員やGoogleが公認したデータ分析エキスパートなど、データアナリティクス分野の専門家が在籍しています。

＜職場の雰囲気＞

仕事に対するプロ意識や学習意欲の高いメンバーが多いです。会社として「頑張っている人はカッコいい」「頑張っている人を応援しよう」という雰囲気があります。

プライベートの時間を大切にするためにも、「スキルを身につける」「テキパキと効率的に仕事をする」「積極的にコミュニケーションを行う」「短時間で成果を出す」といったことを心がけています。
