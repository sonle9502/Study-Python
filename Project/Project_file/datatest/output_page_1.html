【データアナリスト】データ分析を通して郵便局、日本の未来を切り拓く/システムインテグレータ・ソフトハウス - job post

株式会社JPデジタル
東京都 千代田区 大手町 • 一部在宅
年収 600万円 ~ 1,000万円 - 正社員
応募先へ進む前に、Indeed アカウントを作成してください。

職務内容詳細

【データアナリスト】 データの力でオペレーションの効率化や新しい価値の創造を目指す！日本最大級の物流ネットワークのデータ分析をお任せします

事業内容
「みらいの郵便局」の実現に向け、UI/UX、データの分析、最新テクノロジー導入支援、デジタルマーケティング、DX人材育成の5つを軸に、日本郵政グループ内におけるDXを推進。リアルの郵便局ネットワークとデジタルを融合させ、お客さまの体験価値を徹底的に高める「顧客起点のDX」を体現していきます。

仕事内容
JPデジタルのミッションは、リアルとテクノロジーの力を掛け合わせて、これまでにない「みらいの郵便局」を創ることです。今あるものを改善したり、新しいものを生み出すには、データに基づいて現状を正しく把握し、施策を考える必要があります。そうしたデータ駆動の意思決定をグループ全体ですすめるべく、新たなデータ分析チームを立ち上げることになりました。

日本郵政グループでは主に、郵便・物流事業、保険事業、金融事業を行なっていますが、まずは、ユニバーサルサービス（誰もが公平に受けることのできるサービス）を含む郵便・物流事業の課題解決に取り組みます。

今回募集するデータアナリストのミッションは、データ分析を通して、郵便・物流事業のオペレーションや営業活動の効率化、新しい付加価値を導き出すこと。デジタル化された情報を活用して配達員や車両の割り当てを最適化するなどオペレーションを改善するだけでなく、ユーザー（荷物を差し出す人、受け取る人）とのコミュニケーションを変えることで、社会システムの変革をも目指していきます。

物流というリアルな世界では、データを集めるのも、分析結果を試すのも、評価するのも、時間がかかりますし、正直、簡単なことではありません。しかし、オペレーション効率を1%改善するだけで巨額のコスト改善につながる日本最大級の物流ネットワークを最適化するという経験・やりがいは、何にも代え難いものになると思います。

私たちと一緒に、ワクワクするチャレンジをしませんか？データの力で、郵便局の、日本の未来を切り拓いていきましょう！

・具体的な業務内容
日本郵便と協業し、郵便・物流オペレーションに関するデータを分析し、オペレーションや営業活動の効率化、新しい付加価値を創出するための示唆を提供していきます。
また、データ分析チームはこれから立ち上げるチームのため、組織づくりにも積極的に携わっていただければと思います。
＊まずは、郵便・物流に関するデータ分析をお任せしますが、将来的に保険事業や金融事業のデータ分析をお任せする可能性もあります

・業務課題、仮説、施策案をもとにしたデータ分析
・データ分析はGCP、AWSなどパブリッククラウド上で行います
・分析結果をチーム内または日本郵便の担当者と共有し、ディスカッション、分析を繰り返します
・効果的なオペレーション施策や顧客体験デザインの実現に向け、有益な示唆を提供
・分析方法の最適化
・物流ネットワークの可視化、KPIの設計・評価
・実施した施策の定量的な分析・評価
・データ分析チームにおける仕組みづくり、業務フローの整備など

この仕事で得られるもの
・クラウド上でのデータ分析など、最先端の仕組み・環境の中でデータ分析ができる
・たくさんのシステムが協調的に動く社会システムを分析し、最適化していくことで社会を良くしていける
・日本最大級の物流ネットワークというインパクトがある事業に携われる
