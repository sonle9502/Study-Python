職務内容詳細

商品の認知にも販売にも必要不可欠なのが、 高品質な動画クリエイティブ。

動画クリエイターは、商品の広告やPRに必須となった動画の、企画・構成・編集を行い、目的を果たすことができる高品質な動画を作成する業種です。

単に編集作業を行うのではなく、お客様に伝える商品のメッセージを正確に捉え、それを映像としてどう見せるのかを考え、露出するメディアに最適化された編集を行うことを動画クリエイターと定義しています。
