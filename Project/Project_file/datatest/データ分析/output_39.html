NIKE(ナイキ)のMD(アパレル)/ファッション・アパレル人材募集 - job post

株式会社WINWIN YJV
東京都 目黒区 中目黒駅
年収 500万円 ~ 800万円 - 正社員
応募先へ進む前に、Indeed アカウントを作成してください。

勤務地

東京都 目黒区 中目黒駅

職務内容詳細

応募資格
  • 必須条件（いずれも該当）
(1)アパレルブランド・専門店いずれかでのバイヤーまたはMD経験
(2)中級レベル以上のPCスキル（Excel、PowerPointは必須）
(3)数値管理能力
(4)マーケットトレンドなどの分析力
(5)プレゼン能力・コミュニケーション能力

  • 歓迎条件
(1)「NIKE」ブランドのアイテムを扱った経験
(2)「NIKE」ブランドの商品知識
(3)英語または韓国語力

世界的に人気のスニーカー・スポーツアパレルブランドのパートナー企業として設立した同社。
これまでの経験を活かし、ご活躍いただける環境です。

勤務地
東京都目黒区

勤務地最寄り駅
東急東横線 中目黒駅

雇用形態
正社員（試用期間3ヶ月）

想定年収
500万円～800万円

9：00～18：00

無期

雇用保険: 有
労災保険: 有

仕事内容
  • 担当業務
同社にて「NIKE」のMD担当を募集します。
アパレル商品のリテールプランナーポジションをお任せします。

(1)昨シーズンの販売データ分析
(2)消費者動向、マーケットトレンド、競合他社の製品分析などを通じた市場調査・店舗で商品のフィードバック収集
(3)ジェンダー、サイズ、カラー、シルエットなどの分析、昨シーズンのレビュー、シーズン予算の確認
(4)予算に応じたSKU、ジェンダー、カテゴリー別購入準備
(5)「NIKE」側からのシーズン戦略および商品情報のダウンロード
(6)購入戦略プランの作成
(7)商品発注前のサンプル確認およびフィードバック
(8)シーズン入庫CRD（Customer Request Date）配分および決定
(9)最終発注報告書の作成

募集人数
1人

喫煙環境
屋内全面禁煙

事業内容
  • 会社紹介
WINWIN YJVはNIKEの優れた商品やサービスを通して、お客さまの健康と幸せ、生活の質向上を目標とします。
全てのNIKEのお客さまに「スポーツを日常の習慣にする（Make Sport a Daily Habit）」ため、お手伝いします。
様々なサービスを通してNIKEブランドのヘリテージとイノベーションを正確に伝え、NIKEメンバーに満足して頂く事がWINWIN YJVリテール戦略のポイントです。
WINWIN YJVは、NIKEとの強いパートナーシップとNIKEに対する深い理解を持っています。
それらをもとに、デジタルとフィジカルを合わせたNIKEエクスペリエンスを提供することのできる、NIKEメンバーのための最高のNIKEストアを作っていきます。
