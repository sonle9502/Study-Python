職務内容詳細

【概要・業務内容】

私たちは、「人にも食べ物にも幸せな一生を」を掲げ、利用していただける方と社会までプラスに働く仕組みを構築を目指し、日々社会を変えるために取り組んでいます。

今回募集しているのは、一緒に真剣に社会の変革に取り組んでいただく長期インターンです。

インターンの方と社員に権限の差はありません。
”インターン生”でなく、メンバーの一員として真剣に食品や社会の未来変革に取り組んでいただける方を募集しています!

一緒に取り組んでいただく方には、社会性の高い事業の成長ノウハウ,社会で使える汎用的スキルを得る,また当然ですが個人の成長をお約束します。

人間,食品や自然により良い社会を作るべく、私たちの想いに共感して一緒に取り組んで頂ける方是非ご連絡ください!

【rebakeに関して】
rebake(https://rebake.me/)は、全国のおいしいパンをお取り寄せできる通信販売マーケットです。
パンの廃棄を減らす活動も行っており、売れ残ってしまったパンを購入し、パンのロスの削減にも取り組んでいます。
登録店舗は1000件以上(パン屋さんの10%以上),利用者数も10万人以上です。
様々なテレビ等のメディアで取り上げられているサービスです。

主に「rebake」のCS(カスタマーサクセス),顧客満足度向上(UX向上)を担って頂きます。
実際に多くの会員の方に対して手を動かして施策を実行することで、汎用的なスキルが身につきます。
