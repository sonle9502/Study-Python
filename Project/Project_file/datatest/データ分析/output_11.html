業務内容

公立学校教員のメンタルヘルスに関わる業務

  • ストレスチェック制度関連業務（集団分析リポート作成、データ分析）
  • リワークプログラム関連（プログラムの運営、心理治療、心理教育）
  • 心理療法関連（相談業務・心理治療）
  • 各種研修会の講師

当院は公立学校共済組合が運営する公的病院で、公立学校教職員のメンタルヘルス支援事業に力を入れて取り組んでいます。

今回、同事業を拡充させることになり、それに関連する業務に取り組んでいただける心理士の先生方の募集です。関連業務への経験がある方は、そのご経験を活かし、やりがいをもって取り組んでいただけるお仕事だと思います。教職員への心理支援経験がない方でも、経験を積みながら行っていただくことができます。また、一般の方・市民対象のカウンセリングについても、ご希望に応じて、取り組んでいただくことができます。

時間外勤務はほぼ無く、年休も取得しやすく、ワークライフバランスも充実しています。心理士の先生方の専門性を活かしながら、働きがいをもって、できるだけ長くご勤務いただけるような職場環境を提供したいと考えています。ぜひお気軽にご応募ください。
