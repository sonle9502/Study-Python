経営企画(数値管理・採算管理)　~リモート・フレックス有/成長中のfintech事業~

【具体的な仕事内容】
  • 業務内容:
当社の更なる事業拡大により、経営判断に必要な情報をタイムリーに整理し提供することで円滑な企業運営を実現するため、新たな部署を設立することになりました。新部署の立ち上げメンバーとして、0から1を生み出して頂きながら業務構築・推進していただきます。
【主な業務内容】
・社内数値全般の把握および精緻化
・新設部署の業務適性化、アウトプットに向けてのフロー構築
・中期経営計画に関する計数管理・採算管理および要因分析
数値における定量面での社内外情報把握
定性的な要因把握
【変更の範囲: 会社内のすべての業務】
課長や現メンバーとともに上記業務を推進していただき、ゆくゆくは当社の中核となって管理職としてのマネジメントを目指していただけるポジションです。
  • 業務の魅力
拡大フェーズにあるFintech企業にて新部署立ち上げの貴重な経験に携わることが出来るポジションです。
経営視点・全社視点を持ちながら、経営層と近い立ち位置で業務を進めることで会社に貢献していただけます。自身の力で会社全体を動かしていきたい方、新規ビジネスに携わっていきたい方、これまでのご経験を活かしながらキャリアアップを目指されたい方も歓迎いたします!
  • 同社の特徴
・中途入社も馴染みやすい環境
チームワークを大事にする社風で全社員の約9割が中途入社なので、一人ひとりの多様なバックグラウンドを活かし、融合しながら、サービス向上に取り組んでいます。また、オフィスには自由に利用できる落ち着いた雰囲気のカフェスペースもあり、気分を切り替えながら仕事に取り組めます。
・パフォーマンス最大化を目指す就業環境
フルフレックスタイム制を採用し、残業時間も月平均20時間程度なので、メリハリをつけた柔軟な働き方で個人のベストパフォーマンスを発揮できます。従業員のライフステージに対応した、より最適な業務推進環境を目指し、オフィスワークとリモートワークを併用する就業環境をご用意しております。一人ひとりが自律的自発的なパフォーマンスを発揮し、活き活きと働ける環境づくりを推進しております
