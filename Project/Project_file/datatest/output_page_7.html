【具体的な仕事内容】
【大学院卒歓迎/実務経験不問/キユーピー・ライオン・サントリー・大塚製薬・味の素・ソニーミュージックなど大手企業との取引有/シェアNo.1/独自性の高い技術で企業の課題解決を進めます】

  • 同社事業について
同社は「ファンコミュニティクラウド」という独自のサブスクリプション型のソーシャルメディアを用いて顧客のマーケティング課題を解決しています。これまでtoC系の大手企業を中心に、累計200社以上に採用されています。

  • 業務概要【変更の範囲:会社の定める業務】
同社のデータ分析専門チーム(マネジメントを含め女性5名)での、ファンコミュニティに蓄積された消費者の活動データの分析・レポート作成業務をご担当いただきます。

  • 業務詳細
・ご発注いただいたクライアントのコミュニティを運営担当チームが活性させたのち、そこで得られるデータを分析し、企業の意思決定と、ファンコミュニティのPDCAサイクルを支援する分析レポートの作成
※企業向けの報告も担当します。
・どのようなアクションや発言がロイヤリティや購買行動につながるか、当社独自の手法(特許取得済み)を用いたユニークな分析を通して、データによる客観的な事実と消費者の声から情報を抽出
・クライアントを担当するコンサルタントやプランナーなど多様なメンバーと連携し、更なるコミュニティの活性化やマーケティング戦略立案・商品開発・販売促進など様々な領域の事業課題解決に繋げるための支援

  • 魅力
・必須要件に当てはまり当社事業へ共感してくださる方であれば、業務経験不問にてご応募頂けます。
・現在、「企業がいかに消費者に製品を売るのか」といったマーケティングから、「消費者と価値を共に創りだしていく」という時代へと変化しております。その中で過去20年間の消費者コミュニティの運営ノウハウや、国内の大手企業との取引から蓄積されてきたビッグデータや分析ノウハウがある同社。「消費者コミュニティ」のパイオニアとして市場をつくってきた当社は現在、第二創業期として、これまで築き上げてきた土台から発展させるべく、さらなる顧客拡大とグローバル展開を進めています。
・同社の事業は4つの国際的な特許技術を用いたユニークな分析手法を採用しており、業界No.1シェアの実績があります。
