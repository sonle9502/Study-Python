  • 担当いただく役割:

【変更の範囲: 会社の定める業務】

  • 具体的な業務内容:
・顧客の調査レポート(主に観光、インバウンド関連)、マーケティング戦略に資するデータ分析、支援などを行います。
・分析対象とするデータは、人流を把握するための位置情報データ(モバイル空間統計など)、消費に関するデータ、公的統計データ(国土数値情報や国勢調査)などです。
・これらを収集、加工・集計し付加価値の高い図表などに仕上げます。その過程で、R・PythonやExcelなどを適宜用います。また、マップ作成時は、地理情報システム(GIS)を利用します。
・分析対象のデータは、外部のデータベンダーから調達します。顧客(DBJ等)ニーズを踏まえ調達すべきデータの選定や分析手法の提案も行います。

  • 仕事のスタイル、必須要素:
・「DBJとの連携」を基本に、少数精鋭の部署につき、チームワーク・議論・対話・フットワーク重視で運営しています。
・現在は、人流データ、アンケートデータ、消費データが中心ですが、今後、顧客ニーズに応じた多様な新規データ分析を志向していきます。アイデアの整理、提案活動・技術営業も分析と併せて重要な仕事となります。
・データを分析するにあたって、そのデータが持つ意味を理解・推察できる論理的考察力が必要となります。
・個々の案件は中・短期間に、複数案件を同時並行的に対応する場合もあります。
