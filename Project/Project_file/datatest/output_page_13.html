仕事内容

サイトの目標を達成するための施策を立案・検証し、ユーザーの行動データを分析することでサービス開発を支援する業務です。

- サイトデータを分析し、改善点・新機能を考案
- 就活サイトのサービス・機能開発に携わり、ユーザーに価値提供
- マーケティングツールや調査を活用し、サイト利用体験向上施策を実施

求める人材

・データ分析を駆使し、ユーザーの行動把握やサービス改善を主導したい方
・新卒入社で、自社のコンテンツサービス開発に貢献したい方

【必須要件】(あると活かせるご経験)
数値データなどをもとにしたWebサイトの改善・サービス開発に関するご経験
SQLやTableauを使ってWebサイトなどの分析に関するご経験
MicrosoftOffice(Excel/PowerPoint)を業務で操作されたご経験
※ExcelはVLOOKUP関数、ピボットテーブル程度が利用できることが望ましいです

(あると活かせるスキル・マインド)
「仮説→検証→解決策の検討」のプロセスで物事を考える経験や思考をお持ちの方
Web広告やマーケティングについての知識
Web解析ツールのスキル（GoogleAnalyticsなど）
MAツールの知識、運用経験

(こんな方とお会いしたい)
広告運用や解析ツールの使用経験を必須とは考えておらず、サイトやサービスをグロースさせるために、課題・問題点に対して自ら仮説立てから、根拠となる数字を分析、企画・立案などを行ってきた方とお会いしたいと考えています。機能や仕組みを考えられる方、企業と学生のマッチングを追求する就職情報サイトを一緒に成長させていきたい、新たなサービス価値を作っていきたいとお考えの方に是非ご応募いただきたいです。
